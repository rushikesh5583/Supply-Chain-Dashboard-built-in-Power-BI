# Supply ChainDashboard-built-in-Power-BI
This interactive dashboard provides insights into inventory levels, order fulfillment, supplier performance, shipping efficiency, and operational costs. Using dynamic visualizations and KPIs, it helps identify trends, optimize operations, and support data-driven decision-making.

# Key Features:
✔ Inventory Analysis
✔ Order & Sales Tracking
✔ Supplier Performance Monitoring
✔ Shipping & Delivery Insights
✔ Cost Analysis & KPIs
✔ Interactive Filters and Drill-Downs
